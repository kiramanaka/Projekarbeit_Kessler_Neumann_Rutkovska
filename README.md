### Setup
1. Repository klonen
2. In das Projektverzeichnis wechseln
3. `pip install -r requirements.txt`
4. `python run.py`
5. Im Browser `http://127.0.0.1:5000/` öffnen
6. Anmelden mit `admin`/`admin`
7. Viel Spaß beim Testen!
